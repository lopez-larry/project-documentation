## 4. Deployment Steps

* **Pre-deployment Checklist:**

  * Backup databases.
  * Verify infrastructure readiness.
  * Confirm approvals.

* **Deployment Procedure:**

  * Pull latest code from repo.
  * Build and package application.
  * Deploy backend (e.g., Docker on EC2).
  * Deploy frontend (e.g., S3 + CloudFront).
  * Run migrations (DB changes).
  * Apply configurations (secrets, env vars).

* **Post-deployment Verification:**

  * Smoke tests.
  * Confirm key features (login, API calls, payments).

---

## 5. Rollback Plan

* **Rollback Triggers:** Deployment fails, major bug, downtime > threshold.
* **Rollback Steps:**

  * Restore DB backup.
  * Revert code to previous stable release.
  * Notify stakeholders.
